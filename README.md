# Flora - Flower Shop Web Project

A modern, responsive landing page for **Flora**, a boutique flower shop. The project showcases floral arrangements, best-selling bouquets, client feedback, and a fully functional mobile navigation menu.

## Overview
Flora is designed to provide a smooth, elegant, and user-friendly web experience for customers looking to explore and purchase stunning flower bouquets. It features a fully responsive layout optimized for mobile devices, tablets, and desktops.

---

## Features
* **Responsive Navigation:** Sticky header with desktop links and a smooth slide-in mobile menu.
* **Hero Section:** Engaging introductory banner with call-to-action buttons.
* **About Us Section:** Highlights the shop's passion for floral artistry.
* **Bestsellers Carousel/List:** Showcases top-selling items with pricing and navigation dots.
* **Bouquets Catalog:** Grid layout displaying various fresh flower arrangements with a "Show More" interaction.
* **Customer Feedback:** Testimonials section featuring client reviews.
* **Contacts Section:** Direct access to store location, phone numbers, and interactive map links.
* **Footer:** Quick links, social media integration, and copyright information.

## Project Structure
```text
FloraWebProject/
│
├── icons/            # SVG sprites and favicons
├── images/           # Product photos, hero backgrounds, and logos
├── js/               # JavaScript files (e.g., mobile-menu.js)
├── styles/           # CSS stylesheets and modular styles
├── .prettierrc       # Prettier configuration file

```

---

## Technologies Used
* HTML5 & Semantic Markup.
* CSS3 (Flexbox, Grid, Custom Properties/Variables).
* JavaScript (ES6+) for mobile menu and interactive states.
* Modern Normalize for cross-browser consistency.
* Animate.css for animations.

## Setup & Running Locally
1. Clone the repository to your local machine:
```bash
git clone https://github.com/ErnestSosnovskyi/FloraWebProject.git
```
2. Navigate to the project directory:
```bash
cd FloraWebProject
```
3. Open `index.html` in your browser or run it via a local development server (e.g., *Live Server* in *VS Code*).
